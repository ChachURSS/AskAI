# AskAI
Logiciel copilot pour la vie de tout les jours, évidemment que personne va l'utiliser sur theia

## Comment utiliser
Lancer le code.
⬇ Flèche du bas pour masquer/afficher la fenêtre
⬆ Envoyer le texte séléctionner à l'IA
← Flèche gauche pour changer le modèle (20 crédits pas modèle sauf si tu paies)
→ Flèche droite pour le passer en invisible (buggué :'(   sad)

## Mise en place
Aller sur https://aistudio.google.com/app/api-keys
Créer un projet dans l'onglet **Project**
Dans l'onglet **API Key** créer une clé assigné à se projet
Dans **VSCODE** dans le terminal " setx GEMINI_API_KEY "colle-ta-clé-ici" "

## Interactions
Impossibilité de prompt 2 fois le même texte (donc si l'app dit "Pas de texte séléctionné" réessaies en ne séléctionnant pas le numéro de question ou la dernière lettre)
Possibilité de prompt en une fois tout le cctl, mince missclick je voulais dire dans la vie de tout les jours, et utiliser ⬇ pour faire apparaitre la fenêtre quand nécessaire
